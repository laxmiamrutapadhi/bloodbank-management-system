# bloodbank-management-system

## 📌 Steps to Run the Project (Using XAMPP)

1. Install and open **XAMPP Control Panel**  
2. Start **Apache** and **MySQL**

3. Copy your project folder into:
```
C:\xampp\htdocs\
```

4. Open browser and go to:
```
http://localhost/phpmyadmin
```

5. Create a new database (e.g., `bloodbank`)

6. Import the provided `.sql` file into the database

7. Open browser and run the project:
```
http://localhost/your-project-folder-name
```
